# edtla-anon

Anonymous repository accompanying an ACL submission on error-driven targeted LLM augmentation

## EDTLA – Hinglish POS Tagging

### Dataset

The `finalTrain.tsv` and `finalTest.tsv` files in this repository are **placeholders** and do not contain any actual tokens or POS tags in order to comply with Twitter/X policy.

To reproduce our experiments, download the original Hindi–English (Hinglish) Twitter POS tagging corpus:

**Reference:** Singh, A., Sharma, V., & Bhattacharyya, P. (2018). *A Twitter Hindi-English Code-Mixed Corpus for Part-of-Speech Tagging*. Proceedings of COLING 2018.

### Our splits (by token count):

- **Training:** first 29,683 tokens of the dataset
- **Testing:** remaining 3,327 tokens
